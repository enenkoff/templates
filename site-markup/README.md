## install plugins
```
yarn
```
or
```
npm install
```
## gulp build for server
```
gulp build
```

## gulp watcher (default task)
```
gulp watch
```
or
```
gulp
```

## prettier lint
```
yarn prettier
```
or
```
npm run prettier
```

## ESlint (for app-lint.js)
show errors
```
yarn lint
```
try to fix errors
```
yarn lint:write
```
